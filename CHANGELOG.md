# Changelog

このプロジェクトの変更履歴。

バージョン規則:
- **機能追加・バランス調整** → マイナーを上げる（`+0.1`）
- **バグ修正** → パッチを上げる（`+0.0.1`）
- **メジャー** → 指定時のみ

バージョン番号の単一情報源は `godot/project.godot` の `application/config/version`。
変更をマージするたびに、この番号を上げて本ファイルに追記する。

## [0.0.0] - 2026-07-20
### Added
- バージョン管理を導入。単一情報源を `godot/project.godot` の `application/config/version`
  に置き、タイトル / ゲームクリア / ゲームオーバー画面の隅に `vX.Y.Z` を表示するようにした。
